## 🎯 About the Project

The main goal of this website is to serve as a digital business card for the academy. It was structured to answer the main questions of anyone looking to start training Jiu-Jitsu with the team.

### ⚙️ Features:


- **About the Team:** The history of Jungle BJJ Team and its philosophy.
- **Training Schedule:** An organized layout displaying daily classes.
- **Location & Contact:** Visual map integration and direct contact buttons/forms linking to the academy's WhatsApp.
- **Responsive Menu:** Fully optimized navigation for both desktop computers and mobile devices (smartphones and tablets).

---

## 🛠️ Technologies Used

This project was built from scratch, using the core web technologies to ensure lightweight and fast performance:

- ![HTML5](https://img.shields.io/badge/html5-%23E34F26.svg?style=for-the-badge&logo=html5&logoColor=white) **HTML5** - Semantic structure of the website content and sections.
- ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white) **CSS3** - Layout styling, responsiveness (Media Queries), typography, and custom color scheme representing the "Jungle" visual identity.
- ![JavaScript](https://img.shields.io/badge/javascript-%23F7DF1E.svg?style=for-the-badge&logo=javascript&logoColor=black) **JavaScript** - Logic for the mobile hamburger menu toggle, basic form validations, and smooth scroll effects.
